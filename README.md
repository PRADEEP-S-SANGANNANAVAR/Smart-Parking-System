# Smart-Parking-System
mart Car Parking System

About the Project
Finding a parking space in busy areas like malls, colleges, or hospitals is often difficult and time consuming. This project is designed to make parking easier by automatically detecting whether a slot is free or occupied.

The system uses Arduino and ultrasonic sensors to check the presence of vehicles. The result is displayed on an LCD so users can quickly know which slots are available.

Why this Project
In normal parking systems, people spend a lot of time searching for space. This leads to fuel wastage and traffic inside parking areas. This project helps solve these problems by giving real time parking information.

Components Used
Arduino Uno
Ultrasonic Sensors
16 by 2 LCD Display
Potentiometer
Breadboard
Jumper Wires
Power Supply

How it Works
Each parking slot has an ultrasonic sensor that measures the distance.
If a vehicle is present, the distance becomes less and the slot is marked as occupied.
If no vehicle is present, the slot is shown as free.
The Arduino processes this information and displays it on the LCD.
The system keeps running continuously and updates in real time.

Working Flow
Start the system
Initialize Arduino and sensors
Measure distance using sensors
Compare with a set value
Identify slot status
Display result on LCD
Repeat the process

Results
The system can accurately detect vehicles
It shows real time updates
It works continuously without interruption
It consumes low power
